# Social App

Social App Project (Facebook clone) with real time database(firebase) you can chat ,post ,upload images ,and communicate with friends.
features used in the app:
Firebase Core
Firebase Auth
Cloud Firestore
Firebase Storage
Bloc Pattern,
Ui,Ux,
design patern,
Null Safety,
Shared Preference.
Dio,
Light Mode
Dark Mode,
sqflite.
